# libibootim
All the essentials you need when working with Apple iBoot images and PNGs.

## Requirements
- libpng
- CMake

## Installation
Install libibootim with CMake

```bash
git clone https://github.com/XboxOneSogie720/libibootim.git
cd libibootim
mkdir build && cd build
cmake ..
make
sudo make install
```
